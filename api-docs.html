POST /predict_text

Analyze a string of text to determine if it was generated by an AI model (like ChatGPT).

curl -X POST http://localhost:5000/predict_text \ -H "Content-Type: application/json" \ -H "Authorization: Bearer YOUR_API_KEY" \ -d '{"text": "Artificial Intelligence is rapidly evolving..."}'

Response

{ "prediction": "Human Written", "confidence": 85.2, "prob_human": 85.2, "prob_ai": 14.8, "sentences": [ { "text": "Artificial Intelligence is rapidly evolving...", "ai_prob": 0.12 } ] }

POST /predict_video

Upload a video file (MP4/MOV). The system extracts frames and returns an aggregate deepfake probability.

import requests url = "http://localhost:5000/predict_video" headers = {"Authorization": "Bearer YOUR_API_KEY"} files = {"video": open("test_clip.mp4", "rb")} response = requests.post(url, headers=headers, files=files) print(response.json())
